# Optional Scripts

Dieser Ordner ist für spätere optionale Hilfsskripte vorgesehen.

Aktuell enthält das Alpha-Projekt keine benötigten Skripte.

Regeln für spätere Skripte:

- Skripte sind optional und dürfen für den Alpha-Test nicht erforderlich sein.
- Skripte müssen klar dokumentiert werden.
- Wenn Skripte eine eigene Lizenz benötigen, muss diese im Skript oder in diesem Ordner angegeben werden.
- Private Alpha-Daten dürfen nicht automatisch verarbeitet oder veröffentlicht werden.
